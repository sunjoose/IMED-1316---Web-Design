How to do your assignments - from start to finish

Download and extract the Data Files

All assignments begin with data files that match the assignments in the textbook. 

1. Insert your flash drive into your laptop. (See "Materials" in the Syllabus.) 

2. Create a folder on your flash drive. Name it IMED1316 (no spaces).

3. Create a subfolder in IMED1316 called DATAFILES (no spaces)

4. Download the data files from eCampus by clicking here: Data Files 

  • If you get a popup asking where to save them, select the IMED1316 folder on your flash drive.
  • If you do not get a popup asking where to save them, they probably went to your "Downloads" folder. Move them to the IMED1316 folder on your flash drive.

5. Extract the data files. The data files are compressed, so you will need to extract the files. Open File Explorer and find the zipped folder. Right-click (or press and hold) it, select Extract All, and follow the instructions, making sure to select IMED1316 > DATAFILES. 

Download and set up your text editor

Assignments are done using a text editor that you probably do not have. Windows computers come with Notepad or Wordpad, but Notepad++ is what we will use so that I can explain every step to you. 

1. For Windows, click this link: Notepad++. Save it to your IMED1316 folder. If that link doesn't work, try this one: https://portableapps.com/. 

Double click on it and install it to IMED1316.

2. For Mac, click this link: Atom. Drag it to your taskbar.
